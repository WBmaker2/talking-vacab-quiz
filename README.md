# talking-vacab-quiz

## Development

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Deployment

This project is deployed with Vercel.

- Primary live URL: `talking-vacab-quiz.vercel.app`
- Preferred workflow: code change -> `npm run build` -> git commit -> git push -> Vercel deploy
